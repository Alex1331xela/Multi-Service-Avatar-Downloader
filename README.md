# Multi-Service Avatar Downloader
A downloader written in Python that downloads avatars from multiple different services/platforms.

### Currently supported services/platforms:
- Nintendo Miis
    - using a Mii Studio code, Nintendo Network ID, or Nintendo Account ID
    - optional: download a GIF of the Mii spinning
    - optional: download the individual frames that make up the Mii spinning animation
- Roblox
    - using a Roblox account ID
    - optional: download images of the individual outfit pieces each account is currently wearing
